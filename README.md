## Main Components
- ESP8266 (or something similar)
- Transmitter (mine is 433Mhz, so I got WRL-10534)
- Voltage Regulator 3.3V
 - 1000µF lectrolytic Capacitors 
 - 0.1µF Ceramic Capacitor
